# EcoWeb

## Descripción del proyecto

EcoWeb es una página web informativa desarrollada como práctica para la asignatura **Introducción al Desarrollo de Software**.

Su propósito es promover el cuidado del medio ambiente mediante información básica sobre el reciclaje y la correcta clasificación de residuos.

## Integrantes

* Daniel Páez – Líder del proyecto
* Sofía Gómez – Desarrolladora

## Tecnologías utilizadas

* HTML5
* CSS3
* Git
* GitHub

## Funcionalidades implementadas

* Página principal informativa.
* Información sobre la importancia del reciclaje.
* Recomendaciones para reciclar correctamente.
* Tabla de clasificación de residuos.
* Pie de página con información del proyecto.

## Flujo de trabajo

Para el desarrollo del proyecto se utilizó Git y GitHub siguiendo un flujo de trabajo colaborativo basado en ramas independientes, commits descriptivos, Pull Requests y revisión de código antes de integrar los cambios a la rama principal.
